Adding SSH key in Github Profile

  1. Home
  2. Adding SSH key

Linux

  1. Open Terminal.
  2. Check for existing SSH keys. If keys exists, skip next step and continue from step 4. In my case keys exists, but if your are doing it for first time the you will no see the last two line on can get an error: ls -la ~/.ssh
  3. Generate a new SSH key. Press enter whenever it asks for something until you see your shell prompt again: ssh-keygen -t rsa
  4. View newly generated ssh key: cat .ssh/id_rsa.pub
  5. Simply, copy the SSH key to your clipboard. And go to your github account's profile.
  6. Add the SSH key to your GitHub account:
    1. Click on your profile icon in the top right corner of GitHub, then click on "Settings".
    2. In the left sidebar, click on "SSH and GPG keys".
    3. Click on "New SSH key" or "Add SSH key".
    4. Paste your SSH key into the "Key" field and add a descriptive title.
    5. Click "Add SSH key".

Windows

  1. Download and install Git for Windows from git-scm.com.
  2. Open Git Bash from the Start menu.
  3. Check for existing SSH keys(if keys exists, skip next step and continue from step 4):
    ls -al ~/.ssh
  4. Generate a new SSH key:
    ssh-keygen -t rsa
  5. View newly generated ssh key:
    cat .ssh/id_rsa.pub
  6. Simply, copy the SSH key to your clipboard.
  7. Add the SSH key to your GitHub account:
    1. Click on your profile icon in the top right corner of GitHub, then click on "Settings".
    2. In the left sidebar, click on "SSH and GPG keys".
    3. Click on "New SSH key" or "Add SSH key".
    4. Paste your SSH key into the "Key" field and add a descriptive title.
    5. Click "Add SSH key".
